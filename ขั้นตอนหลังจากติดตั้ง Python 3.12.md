# ขั้นตอนหลังจากติดตั้ง Python 3.12

ทำตามลำดับนี้เพื่อให้ RabCheck และ OCR (ดึงเลขเช็คจากรูป) ทำงานได้

---

## 1. ติดตั้ง Python 3.12 (ถ้ายังไม่ได้ติดตั้ง)

- ดาวน์โหลด: https://www.python.org/downloads/release/python-3120/  
  เลือก **Windows installer (64-bit)**
- ติดตั้ง โดยติ๊ก **"Add python.exe to PATH"**
- ปิดแล้วเปิด Command Prompt ใหม่

---

## 2. ติดตั้งแพ็กเกจของ RabCheck ใน Python 3.12

เปิด **Command Prompt** แล้วไปที่โฟลเดอร์ RabCheck:

```text
cd C:\Users\tanat\Documents\RabCheck
```

จากนั้นรัน:

```text
py -3.12 -m pip install -r requirements.txt
```

รอจนขึ้น **Successfully installed ...**

---

## 3. ตรวจสอบว่า OCR พร้อม (ไม่บังคับ แต่แนะนำ)

รันคำสั่ง:

```text
py -3.12 backend\check_ocr.py
```

ถ้าขึ้นครบตามนี้ แปลว่าพร้อมใช้ OCR:

- `[OK] ติดตั้ง pytesseract แล้ว`
- `[OK] เจอ Tesseract ที่: ...`
- `[OK] เวอร์ชัน Tesseract: ...`
- `[OK] มีภาษา Thai (tha) ...`

ถ้ายังมี `[!!]` ดูข้อความแล้วทำตามใน **คู่มือติดตั้ง Tesseract (OCR ภาษาไทย).md**

---

## 4. เปิดระบบ RabCheck

- ดับเบิลคลิกไฟล์ **run.bat** ในโฟลเดอร์ RabCheck  
  (run.bat จะใช้ Python 3.12 ให้อัตโนมัติถ้าติดตั้งไว้แล้ว)
- อย่าปิดหน้าต่าง Command Prompt ที่เปิดขึ้น
- เปิดเบราว์เซอร์ไปที่: **http://localhost:5000**

---

## 5. ใช้งาน (รวมถึงให้ระบบกรอกเลขเช็คอัตโนมัติ)

1. ไปที่แท็บ **อัพโหลดรูป** → อัพโหลดรูปเช็คหรือใบโอนเงิน
2. ไปที่แท็บ **รายการทั้งหมด** → คลิก **"+ เพิ่มรายการ (กรอกเอง)"**
3. ฟอร์มจะเปิดพร้อมรูปที่อัพโหลด — ถ้า OCR ทำงาน **ช่อง "เช็คเลขที่" จะถูกเติมเลขจากรูปให้อัตโนมัติ** (เช่น 88063930)
4. ตรวจ/แก้ข้อมูลอื่น (เล่มที่, เลขที่ IV, จำนวนเงิน ฯลฯ) แล้วคลิก **บันทึก**

---

| 3 | (แนะนำ) รัน `py -3.12 backend\check_ocr.py` เพื่อตรวจ OCR |
