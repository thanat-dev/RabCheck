# คู่มือรัน RabCheck แบบ Offline บน Windows พร้อมไอคอน

คู่มือนี้อธิบายวิธีให้ RabCheck รันแบบ Offline บนเครื่อง มีไอคอนชื่อ "RabCheck" กดแล้วใช้งานได้ทันที และยังส่งไป Google Sheet ได้เมื่อต้องการ (ต้องเชื่อมอินเทอร์เน็ต)

---

## สรุปการทำงาน

| คุณสมบัติ | รองรับ |
|-----------|--------|
| ใช้งาน Offline (บันทึกเช็ค, รายการ, สรุปรายงาน) | ✅ |
| เปิดด้วยไอคอน RabCheck | ✅ |
| กดแล้วเปิดเบราว์เซอร์ใช้ได้เลย | ✅ |
| ส่งไป Google Sheet (เมื่อเชื่อมเน็ต) | ✅ |

---

## แนวทางที่ 1: สร้าง Shortcut พร้อมไอคอน (แนะนำ)

### ขั้นตอน

#### 1. สร้างไฟล์ Shortcut

1. ไปที่โฟลเดอร์ `RabCheck` (โฟลเดอร์โปรเจกต์)
2. คลิกขวาที่ `RabCheck_Launcher.py` → **ส่งไปยัง** → **Desktop (สร้างทางลัด)**
3. เปลี่ยนชื่อทางลัดเป็น **RabCheck**
4. (ถ้าต้องการไอคอน) คลิกขวาที่ทางลัด → **คุณสมบัติ** → แท็บ **ทางลัด** → คลิก **เปลี่ยนไอคอน...**
   - เลือกไอคอนที่ต้องการ หรือวาง path ของไฟล์ `.ico` เช่น `C:\Users\[ชื่อคุณ]\Documents\RabCheck\RabCheck.ico`
   - กด **ตกลง** → **ตกลง**

#### 2. ตั้งค่าเป้าหมายของ Shortcut

1. คลิกขวาที่ทางลัด → **คุณสมบัติ**
2. ช่อง **เป้าหมาย** ใส่หนึ่งในแบบต่อไปนี้ (แก้ path ให้ตรงกับเครื่องของคุณ):

   **แบบ ก – ใช้ Launcher (แนะนำ, ไม่แสดงหน้าต่าง Command):**
   ```
   pythonw "C:\Users\[ชื่อคุณ]\Documents\RabCheck\RabCheck_Launcher.py"
   ```

   **แบบ ข – ใช้ run_with_browser.bat (แสดงหน้าต่าง Command):**
   ```
   C:\Users\[ชื่อคุณ]\Documents\RabCheck\run_with_browser.bat
   ```

   **แบบ ค – ใช้ cmd + py (ถ้า pythonw ใช้ไม่ได้):**
   ```
   C:\Windows\System32\cmd.exe /c cd /d "C:\Users\[ชื่อคุณ]\Documents\RabCheck" && py -3.12 RabCheck_Launcher.py
   ```

3. ช่อง **เริ่มต้นที่** ใส่:
   ```
   C:\Users\[ชื่อคุณ]\Documents\RabCheck
   ```

4. กด **ตกลง**

#### 3. ทดสอบ

- ดับเบิลคลิกไอคอน **RabCheck** บน Desktop
- รอ 2–3 วินาที เบราว์เซอร์จะเปิดไปที่ RabCheck อัตโนมัติ
- ใช้งานได้ตามปกติ
- **ส่งไป Google Sheet**: เมื่อต้องการ ส่งไป Google Sheet คลิกปุ่มดังกล่าว (ต้องเชื่อมอินเทอร์เน็ตช่วงนั้น)

**หมายเหตุ:** ถ้าใช้แบบ ก (pythonw) จะไม่มีหน้าต่าง Command — เมื่อใช้งานเสร็จ ให้กด Ctrl+Shift+Esc เปิด Task Manager แล้วปิด process ชื่อ "Python" หรือ "pythonw" เพื่อหยุดเซิร์ฟเวอร์

---

## แนวทางที่ 2: ใช้ run.bat (แสดงหน้าต่าง Command)

ถ้าต้องการเห็นหน้าต่าง Command เพื่อดูสถานะเซิร์ฟเวอร์:

1. ปรับ `run.bat` ให้เปิดเบราว์เซอร์อัตโนมัติ (ดูไฟล์ `run_with_browser.bat` ด้านล่าง)
2. สร้าง Shortcut ไปที่ `run.bat` หรือ `run_with_browser.bat`
3. เปลี่ยนชื่อเป็น **RabCheck** และใส่ไอคอนตามต้องการ

---

## สร้างไฟล์ไอคอน RabCheck.ico

ถ้าต้องการไอคอนเป็นของตัวเอง:

1. หาไฟล์ `.ico` จากเว็บ (เช่น [flaticon.com](https://www.flaticon.com) ค้น "check" หรือ "document")
2. บันทึกเป็น `RabCheck.ico` ไว้ในโฟลเดอร์โปรเจกต์ (`C:\Users\[ชื่อคุณ]\Documents\RabCheck\`)
3. ใน Shortcut: คลิกขวา → คุณสมบัติ → เปลี่ยนไอคอน → เรียกดู → เลือก `RabCheck.ico`

**ตัวเลือกอื่น:** ใช้ไอคอน built-in ของ Windows เช่น `%SystemRoot%\System32\shell32.dll` แล้วเลือกไอคอนรูปโฟลเดอร์หรือเอกสาร

---

## การส่งไป Google Sheet (เมื่อเชื่อมเน็ต)

- ระบบทำงาน Offline ได้ทั้งหมด (อัพโหลดรูป, บันทึก, แก้ไข, สรุปรายงาน, ส่งออก CSV)
- **ส่งไป Google Sheet** ใช้เฉพาะเมื่อต้องการ → เปิดอินเทอร์เน็ต คลิกปุ่ม "ส่งไป Google Sheet" ตามคู่มือ "คู่มือส่งข้อมูลไป Google Sheet.md"
- ถ้า Offline ปุ่มนี้จะไม่ทำงานจนกว่าจะเชื่อมเน็ต

---

## หมายเหตุ

- ต้องติดตั้ง **Python** และ dependencies ตามคู่มือการติดตั้ง
- ถ้าใช้ OCR ต้องติดตั้ง **Tesseract** แยก
- ไฟล์ `rabcheck.db` และโฟลเดอร์ `uploads` อยู่ในโฟลเดอร์โปรเจกต์ — อย่าย้ายหรือลบ
