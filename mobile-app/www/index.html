ระบบบันทึกเช็คและรายการใบเสร็จ (RabCheck)

แอปจะโหลดหน้าจอจากเซิร์ฟเวอร์ RabCheck โดยอัตโนมัติ

ถ้าเห็นหน้านี้ แสดงว่าแอปยังโหลดเซิร์ฟเวอร์ไม่ได้ — กรุณาตรวจสอบ:

  • เซิร์ฟเวอร์ RabCheck รันอยู่บนคอมแล้วหรือยัง (run.bat)
  • มือถืออยู่ WiFi เดียวกับคอมหรือไม่ (ถ้าใช้ IP ในเครือข่าย)
  • ค่า server.url ใน capacitor.config.json ถูกต้องหรือไม่ — แก้แล้วต้องรัน npx cap sync แล้ว build APK ใหม่
