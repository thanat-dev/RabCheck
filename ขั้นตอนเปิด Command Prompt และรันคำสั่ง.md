# ขั้นตอนเปิด Command Prompt ที่โฟลเดอร์ RabCheck แล้วรันคำสั่ง

ทำตามทีละขั้นด้านล่าง

---

## ขั้นที่ 1: เปิดโฟลเดอร์ RabCheck

1. กดปุ่ม **Windows** บนคีย์บอร์ด (หรือคลิกปุ่ม Start)
2. พิมพ์คำว่า **File Explorer** หรือ **Explorer** แล้วกด Enter (หรือเปิด "This PC" / "คอมพิวเตอร์ของฉัน")
3. ไปที่โฟลเดอร์ที่เก็บโปรเจกต์ เช่น:
   - **เอกสาร (Documents)** → โฟลเดอร์ **RabCheck**
   - หรือ path เต็ม เช่น `C:\Users\FINANCE-DPF-NEW-1\Documents\RabCheck`
4. ตรวจสอบว่าในโฟลเดอร์นี้มีไฟล์ **requirements.txt** และโฟลเดอร์ **backend**, **frontend** อยู่จริง

---

## ขั้นที่ 2: เปิด Command Prompt ที่โฟลเดอร์นี้

เลือกทำ **อย่างใดอย่างหนึ่ง** จากสองวิธีด้านล่าง

### วิธี A (ง่ายที่สุด)

1. อยู่ที่โฟลเดอร์ **RabCheck** ใน File Explorer
2. ดูที่ **แถบที่อยู่ (Address bar)** ด้านบน — เป็นช่องที่แสดง path ปัจจุบัน (เช่น `C:\Users\...\RabCheck`)
3. **คลิกหนึ่งครั้ง** ที่ช่องแถบที่อยู่นั้น (path จะถูกเลือกทั้งเส้น)
4. พิมพ์คำว่า **cmd** (พิมพ์ทับ path เดิมได้เลย)
5. กด **Enter**
6. จะมีหน้าต่าง **Command Prompt** (พื้นหลังดำ/เทา) เปิดขึ้น และโฟลเดอร์ปัจจุบันจะอยู่ที่ **RabCheck** แล้ว

### วิธี B (ใช้คำสั่ง cd)

1. กดปุ่ม **Win + R** พร้อมกัน (หรือคลิก Start แล้วพิมพ์ **run** แล้วเปิด "Run")
2. ในช่อง "Open" พิมพ์: **cmd** แล้วกด **Enter**
3. หน้าต่าง Command Prompt จะเปิด (โฟลเดอร์เริ่มต้นอาจไม่ใช่ RabCheck)
4. พิมพ์คำสั่งด้านล่างนี้ (ถ้าโฟลเดอร์ RabCheck อยู่ที่อื่น ให้แก้ path ให้ตรงกับเครื่องคุณ):

   ```text
   cd /d C:\Users\FINANCE-DPF-NEW-1\Documents\RabCheck
   ```

5. กด **Enter**
6. พิมพ์คำสั่งด้านล่างเพื่อตรวจว่าเข้าโฟลเดอร์ถูกแล้ว:

   ```text
   dir
   ```

   กด **Enter** — ควรเห็นรายการไฟล์รวมถึง **requirements.txt** และโฟลเดอร์ **backend**, **frontend**

**สำคัญ:** ก่อนรัน `pip install` ให้ดูที่บรรทัดแรกของ Command Prompt ต้องลงท้ายด้วย **\RabCheck>** เช่น  
`C:\Users\FINANCE-DPF-NEW-1\Documents\RabCheck>`  
ถ้ายังเป็นแค่ `C:\Users\FINANCE-DPF-NEW-1>` แสดงว่ายังไม่อยู่ในโฟลเดอร์ RabCheck ต้องใช้คำสั่ง `cd` เข้าไปก่อน

---

## ขั้นที่ 3: รันคำสั่งติดตั้งแพ็กเกจ

ใน Command Prompt ที่โฟลเดอร์ RabCheck แล้ว:

1. พิมพ์คำสั่งด้านล่างนี้ (แนะนำให้ใช้คำสั่งนี้เลย จะได้ไม่เจอปัญหา "pip ไม่รู้จัก"):

   ```text
   python -m pip install -r requirements.txt
   ```

2. กด **Enter**
3. รอจนมีข้อความประมาณ **Successfully installed ...** แสดงที่ด้านล่าง แสดงว่าติดตั้งครบแล้ว

**ถ้าขึ้นว่า "python ไม่รู้จัก" หรือ "Python was not found"** แสดงว่ายังไม่ได้ติดตั้ง Python หรือไม่ได้ติ๊ก "Add python.exe to PATH" ตอนติดตั้ง — ให้ไปติดตั้ง Python ใหม่และติ๊กช่องนั้น หรือดูคู่มือ "คู่มือการติดตั้งและใช้งาน.md" ส่วนที่ 2

---

## ขั้นที่ 4: (เมื่อจะรันระบบ) รันเซิร์ฟเวอร์

ใน Command Prompt ที่โฟลเดอร์ RabCheck เหมือนเดิม:

1. พิมพ์คำสั่ง:

   ```text
   python backend\app.py
   ```

2. กด **Enter**
3. ถ้ารันสำเร็จ จะมีข้อความประมาณ **Running on http://127.0.0.1:5000**
4. **อย่าปิดหน้าต่าง Command Prompt นี้** เปิดเบราว์เซอร์ไปที่ **http://localhost:5000** เพื่อใช้ระบบ

หรือแทนที่จะพิมพ์คำสั่งในขั้นที่ 4 คุณสามารถ **ดับเบิลคลิกไฟล์ run.bat** ในโฟลเดอร์ RabCheck ได้เลย (จะรัน `python backend\app.py` ให้อัตโนมัติ)

---

## สรุปสั้นๆ

| ลำดับ | ทำอะไร |
|-------|--------|
| 1 | เปิดโฟลเดอร์ **RabCheck** ใน File Explorer |
| 2 | ที่แถบที่อยู่ด้านบน พิมพ์ **cmd** แล้วกด Enter (หรือเปิด cmd แล้วใช้คำสั่ง `cd /d ...\RabCheck`) |
| 3 | พิมพ์ **pip install -r requirements.txt** แล้วกด Enter |
| 4 | เมื่อจะใช้ระบบ พิมพ์ **python backend\app.py** แล้วกด Enter หรือดับเบิลคลิก **run.bat** |

ถ้าทำครบ 4 ขั้นแล้ว ระบบจะพร้อมใช้งานที่ **http://localhost:5000**
