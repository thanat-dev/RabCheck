# คู่มือการติดตั้งและใช้งานระบบ RabCheck แบบละเอียด

คู่มือนี้จะพาคุณติดตั้งและเริ่มใช้งานระบบบันทึกเช็คและรายการจ่ายเงินตั้งแต่ต้นจนเปิดใช้งานได้

---

## ส่วนที่ 1: สิ่งที่ต้องเตรียม

- คอมพิวเตอร์ Windows 10 หรือใหม่กว่า
- การเชื่อมต่ออินเทอร์เน็ต (ใช้ตอนดาวน์โหลด Python และแพ็กเกจ)
- โฟลเดอร์โปรเจกต์ **RabCheck** (ที่มีไฟล์ `requirements.txt` และโฟลเดอร์ `backend`, `frontend` อยู่ภายใน)

---

## ส่วนที่ 2: ติดตั้ง Python

ระบบ RabCheck ใช้ Python 3.8 ขึ้นไป ถ้ายังไม่มี Python ให้ทำตามขั้นตอนด้านล่าง

### ขั้นที่ 2.1 ดาวน์โหลด Python

1. เปิดเบราว์เซอร์ไปที่: **https://www.python.org/downloads/**
2. คลิกปุ่ม **"Download Python 3.x.x"** (ตัวเลขเวอร์ชันล่าสุด)
3. รอให้ไฟล์ติดตั้ง (เช่น `python-3.12.x-amd64.exe`) ดาวน์โหลดเสร็จ

### ขั้นที่ 2.2 ติดตั้ง Python บน Windows

1. ดับเบิลคลิกไฟล์ที่ดาวน์โหลดมา
2. **สำคัญ:** หน้าแรกของตัวติดตั้ง ให้ติ๊กถูกที่ช่อง  
   **"Add python.exe to PATH"**  
   แล้วคลิก **"Install Now"**
3. ถ้ามีข้อความถามสิทธิ์ (User Account Control) ให้คลิก **"Yes"**
4. รอจนติดตั้งเสร็จ แล้วคลิก **"Close"**
5. **ปิด Command Prompt หรือ PowerShell เก่าทั้งหมด** แล้วเปิดหน้าต่างใหม่ (เพื่อให้ระบบรู้คำสั่ง `python` และ `pip`)

### ขั้นที่ 2.3 ตรวจสอบว่าติดตั้ง Python สำเร็จ

1. กด **Win + R** พิมพ์ **cmd** แล้วกด Enter (เปิด Command Prompt)
2. พิมพ์คำสั่งด้านล่างแล้วกด Enter:

   ```text
   python --version
   ```

3. ถ้าติดตั้งถูกต้อง จะเห็นข้อความประมาณ  
   **Python 3.12.x** (หรือเวอร์ชันที่คุณติดตั้ง)  
   ถ้าเห็นคำว่า "ไม่รู้จักคำสั่ง" หรือ "not recognized" แสดงว่า Python ยังไม่อยู่ใน PATH ให้กลับไปติดตั้งใหม่และติ๊ก **"Add python.exe to PATH"**
4. ตรวจสอบคำสั่ง `pip` ด้วย:

   ```text
   pip --version
   ```

   ควรเห็นข้อความเกี่ยวกับ pip และ path ของ Python

## ส่วนที่ 3: ติดตั้งโปรเจกต์ RabCheck

### ขั้นที่ 3.1 เปิด Command Prompt ที่โฟลเดอร์ RabCheck

**วิธีที่ 1 (แนะนำ)**  
1. เปิด File Explorer ไปที่โฟลเดอร์ **RabCheck** (เช่น `C:\Users\FINANCE-DPF-NEW-1\Documents\RabCheck`)
2. ที่แถบที่อยู่ด้านบน คลิกที่ช่อง path แล้วพิมพ์ **cmd** แล้วกด Enter  
   หน้าต่าง Command Prompt จะเปิดโดยโฟลเดอร์ปัจจุบันคือโฟลเดอร์ RabCheck

**วิธีที่ 2**  
1. เปิด Command Prompt (Win + R → พิมพ์ **cmd** → Enter)
2. พิมพ์คำสั่ง (แก้ path ให้ตรงกับที่เก็บโฟลเดอร์ RabCheck ของคุณ):

   ```text
   cd C:\Users\FINANCE-DPF-NEW-1\Documents\RabCheck
   ```

   แล้วกด Enter

### ขั้นที่ 3.2 ติดตั้งแพ็กเกจที่ระบบใช้

ใน Command Prompt ที่อยู่ที่โฟลเดอร์ RabCheck แล้ว พิมพ์:

```text
pip install -r requirements.txt
```

กด Enter แล้วรอจนติดตั้งครบ (จะเห็นข้อความ Successfully installed ...)

- ถ้าใช้คำสั่ง `pip` ไม่ได้ ให้ลองใช้:

  ```text
  python -m pip install -r requirements.txt
  ```

### ขั้นที่ 3.3 ตรวจสอบว่าติดตั้งสำเร็จ

พิมพ์:

```text
python -c "import flask; print('OK')"
```

ถ้าแสดง **OK** แสดงว่าติดตั้ง Flask และ dependencies พื้นฐานเรียบร้อย

## ส่วนที่ 4: (ไม่บังคับ) ติดตั้ง Tesseract สำหรับดึงข้อความจากรูป

ถ้าต้องการให้ระบบ **ดึงข้อความจากรูปเช็ค/ใบโอนเงินอัตโนมัติ** (OCR ภาษาไทย รวมถึงเลขเช็คอัตโนมัติ) ต้องติดตั้ง Tesseract เพิ่ม

- **ถ้าไม่ติดตั้ง:** ระบบยังใช้ได้ปกติ แค่ต้อง **กรอกข้อมูลเองทั้งหมด** หลังอัพโหลดรูป
- **คู่มือติดตั้งแบบละเอียด:** ดูในไฟล์ **คู่มือติดตั้ง Tesseract (OCR ภาษาไทย).md** ในโฟลเดอร์โปรเจกต์

### ขั้นที่ 4.1 ดาวน์โหลด Tesseract สำหรับ Windows

1. เปิดเบราว์เซอร์ไปที่: **https://github.com/UB-Mannheim/tesseract/wiki**
2. ดาวน์โหลดตัวติดตั้งล่าสุด (เช่น **tesseract-ocr-w64-setup-5.x.x.exe**)
3. จำ path ที่ติดตั้ง (ปกติคือ `C:\Program Files\Tesseract-OCR`)

### ขั้นที่ 4.2 ติดตั้ง Tesseract

1. ดับเบิลคลิกไฟล์ที่ดาวน์โหลด
2. ติดตั้งตามขั้นตอน (Next ไปเรื่อยๆ ได้)
3. **สำคัญ:** ในขั้น "Choose Components" ให้ติ๊กเลือก **"Additional language data"** แล้วเลือก **Thai** ด้วย
4. ติดตั้งจนเสร็จ

### ขั้นที่ 4.3 เพิ่ม Tesseract เข้า PATH (ถ้าต้องการ)

ถ้าหลังติดตั้งแล้วรันระบบแล้ว OCR ยังไม่ทำงาน อาจต้องเพิ่ม path ของ Tesseract:

1. กด **Win** พิมพ์ **environment** แล้วเลือก **"Edit the system environment variables"**
2. คลิก **"Environment Variables..."**
3. ในส่วน **"System variables"** เลือก **Path** แล้วคลิก **"Edit"**
4. คลิก **"New"** แล้วเพิ่ม path เช่น  
   `C:\Program Files\Tesseract-OCR`
5. กด **OK** ทุกหน้าต่าง แล้วปิด Command Prompt เก่าและเปิดใหม่

## ส่วนที่ 5: เริ่มรันระบบ

### วิธีที่ 1: ใช้ไฟล์ run.bat (ง่ายที่สุด)

1. เปิด File Explorer ไปที่โฟลเดอร์ **RabCheck**
2. ดับเบิลคลิกไฟล์ **run.bat**
3. จะมีหน้าต่าง Command Prompt เปิดขึ้น และมีข้อความประมาณ  
   **Starting RabCheck server at http://localhost:5000**
4. **ห้ามปิดหน้าต่างนี้** ระหว่างใช้งาน ปล่อยไว้แล้วไปเปิดเบราว์เซอร์

### วิธีที่ 2: ใช้ Command Prompt

1. เปิด Command Prompt แล้วไปที่โฟลเดอร์ RabCheck (ตามขั้นที่ 3.1)
2. พิมพ์:

   ```text
   python backend\app.py
   ```

3. กด Enter จะเห็นข้อความประมาณ  
   **Running on http://127.0.0.1:5000**  
4. อย่าปิดหน้าต่าง Command Prompt นี้

### เปิดหน้าเว็บระบบ

1. เปิดเบราว์เซอร์ (Chrome, Edge, Firefox ฯลฯ)
2. ที่ช่องที่อยู่ (Address bar) พิมพ์:

   ```text
   http://localhost:5000
   ```

3. กด Enter จะเห็นหน้า **"ระบบบันทึกเช็คและรายการจ่ายเงิน"** พร้อมเมนู อัพโหลดรูป, รายการทั้งหมด, สรุปรายงาน

### เปิดในมือถือ (มือถือและคอมต้องอยู่ใน WiFi เดียวกัน)

1. **เปิดระบบบนคอม** — ดับเบิลคลิก `run.bat` (หรือรัน `python backend\app.py`) แล้วปล่อยหน้าต่างไว้
2. **หา IP ของคอม** — บนคอมที่รันระบบอยู่:
   - กด **Win + R** พิมพ์ **cmd** แล้วกด Enter
   - พิมพ์ `ipconfig` แล้วกด Enter
   - ดูที่ **"IPv4 Address"** ของส่วน **"Wireless LAN adapter Wi-Fi"** หรือ **"Ethernet adapter"** (เช่น `192.168.1.105`)
3. **เปิดเบราว์เซอร์บนมือถือ** (Chrome, Safari ฯลฯ)
4. พิมพ์ที่ช่องที่อยู่:

   ```text
   http://192.168.1.105:5000
   ```

   (เปลี่ยนตัวเลขให้ตรงกับ IP ของคอมคุณ)
5. กด Enter จะเห็นระบบเหมือนบนคอม — สามารถอัพโหลดรูปได้เลย (ถ่ายรูปแล้วอัพโหลด หรือเลือกรูปจากแกลเลอรี่)

> **หมายเหตุ:** มือถือและคอมพิวเตอร์ต้อง**เชื่อมต่อ WiFi เดียวกัน** ถ้าเข้าไม่ได้ ให้ตรวจสอบว่า firewall บนคอมไม่ได้บล็อกพอร์ต 5000
> **แนวทางแปลงเป็นแอปมือถือ:** ดูรายละเอียดในไฟล์ **คู่มือแนวทางแปลงเป็นแอปมือถือ.md** (PWA ติดตั้งบนหน้าจอ, ห่อเป็น APK, ฯลฯ)

### เปิดจากเน็ตข้างนอก (มือถือใช้เน็ตมือถือ หรือต่างที่กัน)

ถ้าต้องการเปิดจาก**นอกเครือข่าย** (เช่น มือถือใช้เน็ตมือถือ หรืออยู่คนละที่กับคอม) ต้องใช้ **tunnel** เพื่อให้คน outside เข้าถึงเครื่องคุณได้

**วิธีที่แนะนำ: ใช้ ngrok (ฟรี)**

1. **ดาวน์โหลด ngrok**  
   - ไปที่ https://ngrok.com/download  
   - ดาวน์โหลดสำหรับ Windows แล้วแตกไฟล์

2. **สมัครบัญชี (ฟรี)**  
   - สมัครที่ https://ngrok.com  
   - หลังสมัครจะได้ **authtoken** มาอยู่ที่หน้า dashboard  
   - **ตั้งค่า authtoken (ทำครั้งเดียว):** เปิด Command Prompt ที่โฟลเดอร์ ngrok แล้วรันคำสั่งด้านล่าง (ใส่ token ของคุณแทน `YOUR_TOKEN`)

     **ขั้นตอนโดยละเอียด:**

     1. **เปิด Command Prompt**  
        - กด `Win + R` พิมพ์ `cmd` แล้วกด Enter  
        - หรือเปิด Start Menu พิมพ์ "Command Prompt" แล้วคลิกเปิด

     2. **ไปที่โฟลเดอร์ที่แตกไฟล์ ngrok**  
        - โฟลเดอร์หลังแตกอาจชื่อ `ngrok-v3`, `ngrok` หรืออื่นๆ — ให้เปิดโฟลเดอร์นั้นใน File Explorer แล้วดู path จริง  
        - ตัวอย่าง: ถ้าอยู่ใน Downloads และโฟลเดอร์ชื่อ `ngrok` ให้พิมพ์:
          ```text
          cd C:\Users\ชื่อคุณ\Downloads\ngrok
          ```
        - ถ้าอยู่ใน Downloads อยู่แล้ว แค่มีโฟลเดอร์ย่อยชื่อ `ngrok` ให้พิมพ์: `cd ngrok`  
        - กด Enter  
        - **หมายเหตุ:** คำสั่งที่ต้องรันคือ `ngrok` (ชื่อโปรแกรม) ไม่ใช่ชื่อโฟลเดอร์ เช่น อย่าพิมพ์ `ngrok-v3` เป็นคำสั่ง

     3. **ใส่ token**  
        - ไปที่ https://dashboard.ngrok.com/get-started/your-authtoken  
        - คัดลอก **authtoken** (เป็นสายอักขระยาว)  
        - ที่ Command Prompt พิมพ์ (คำสั่งคือ **ngrok** แล้วตามด้วย config add-authtoken และ token):
          ```text
          ngrok config add-authtoken วาง_TOKEN_ที่คัดลอกไว้ตรงนี้
          ```
        - ตัวอย่าง (token เป็นตัวอย่างเท่านั้น):  
          `ngrok config add-authtoken 2abc3def4ghi5jkl_6mnoPQRstuVWXyz`

     4. **กด Enter**  
        - ถ้าสำเร็จจะขึ้นข้อความประมาณว่า "Authtoken saved to configuration file"  
        - ตั้งค่านี้**ครั้งเดียว** ครั้งต่อไปไม่ต้องรันคำสั่งนี้อีก

     ```text
     ngrok config add-authtoken YOUR_TOKEN
     ```

3. **รันระบบ RabCheck**  
   - ดับเบิลคลิก `run.bat` ให้ระบบทำงานอยู่

4. **เปิด tunnel**  
   - เปิด Command Prompt อีกหน้าต่างหนึ่ง  
   - ไปที่โฟลเดอร์ ngrok แล้วรัน:

     ```text
     ngrok http 5000
     ```

5. **ใช้ URL ที่ได้**  
   - ngrok จะแสดง URL เช่น `https://abc123.ngrok-free.app`  
   - **เปิด URL นี้บนมือถือหรือเครื่องใดก็ได้** (แม้ใช้เน็ตข้างนอก) — จะเข้าได้เหมือนปกติ

> **หมายเหตุ:**  
> - URL ของ ngrok ฟรีจะ**เปลี่ยนทุกครั้ง**ที่รันใหม่  
> - ตอนรัน ngrok อย่าปิดหน้าต่างนั้น  
> - ข้อมูลส่งผ่าน ngrok ถูกเข้ารหัส (HTTPS)  
> - ใช้เฉพาะเมื่อต้องการเท่านั้น เพราะเปิดระบบให้เข้าจากอินเทอร์เน็ตได้

**เปิด ngrok ด้วยไอคอนครั้งเดียว (แสดง Command Prompt ทันที)**

- ในโฟลเดอร์ RabCheck มีไฟล์ **`run_ngrok.bat`**  
- ดับเบิลคลิก `run_ngrok.bat` → จะเปิดหน้าต่าง Command Prompt แล้วรันคำสั่งตั้งค่า authtoken และ `ngrok http 5000` ให้เอง  
- ถ้าโฟลเดอร์ที่เก็บไฟล์ `ngrok.exe` ไม่ใช่โฟลเดอร์ **Downloads** ให้เปิด `run_ngrok.bat` ด้วย Notepad แล้วแก้บรรทัด `set "NGROK_DIR=..."` ให้ชี้ไปที่โฟลเดอร์ที่มี `ngrok.exe`  
- **สร้าง Shortcut เป็นไอคอน:** คลิกขวาที่ `run_ngrok.bat` → **ส่งไปที่** → **เดสก์ท็อป (สร้างทางลัด)** จากนั้นจะได้ไอคอนบน Desktop ไว้คลิกเปิด ngrok ได้ครั้งเดียว

## ส่วนที่ 6: วิธีใช้งานระบบ

### 6.1 อัพโหลดรูปเช็คหรือใบโอนเงิน

1. ที่หน้าแรก เลือกแท็บ **"อัพโหลดรูป"**
2. **วิธีอัพโหลด:**
   - **จากมือถือ:** ส่งรูปจากมือถือมาที่คอม (ไลน์, อีเมล, หรือ copy ลงโฟลเดอร์ในคอม) แล้วลากไฟล์รูปมาวางในกรอบ "ลากรูปมาวาง หรือ คลิกเพื่อเลือกรูป"
   - **จากคอม:** คลิกในกรอบสีเทา แล้วเลือกไฟล์รูปจากเครื่อง (รองรับ .png, .jpg, .jpeg, .gif, .webp)
3. หลังอัพโหลด ระบบจะแสดงข้อความว่าอัพโหลดสำเร็จ และถ้ามีการติดตั้ง Tesseract จะพยายามดึงข้อมูลจากรูปให้
4. แนะนำให้ไปที่ **"รายการทั้งหมด"** แล้วคลิก **"แก้ไข"** ที่รายการที่เพิ่งเพิ่ม เพื่อตรวจและแก้ไขข้อมูลที่ดึงได้ (หรือกรอกเพิ่มถ้าดึงไม่ได้)

### 6.2 ดูและแก้ไขรายการ (รายการทั้งหมด)

1. คลิกแท็บ **"รายการทั้งหมด"**
2. จะเห็นตารางรายการทั้งหมด (วัน/เดือน/ปี, เล่มที่, เลขที่ IV, เช็คเลขที่, จำนวนเงิน, ชื่อผู้ซื้อ/สถานที่ ฯลฯ)
3. **แก้ไขรายการ:** คลิกปุ่ม **"แก้ไข"** ที่แถวที่ต้องการ จะมีหน้าต่างฟอร์มเปิดขึ้น
   - แก้ไขข้อมูลในช่องต่างๆ ให้ตรงกับเช็ค/ใบโอนเงิน
   - ถ้ามีรูป จะแสดงรูปด้านบนฟอร์ม
   - คลิก **"บันทึก"** เมื่อแก้เสร็จ
4. **เพิ่มรายการเอง (ไม่ต้องอัพโหลดรูป):** คลิกปุ่ม **"+ เพิ่มรายการ (กรอกเอง)"** แล้วกรอกข้อมูลในฟอร์ม จากนั้นคลิก **"บันทึก"**
5. **ลบรายการ:** เปิดฟอร์มแก้ไขรายการนั้น แล้วคลิกปุ่ม **"ลบรายการ"** และยืนยัน

### 6.3 สรุปรายงานและพิมพ์

1. คลิกแท็บ **"สรุปรายงาน"**
2. ระบบจะโหลดข้อมูลทั้งหมดมาแสดงเป็นตารางสรุป (ลำดับ, วัน/เดือน/ปี, เล่มที่, เลขที่ IV, เช็คเลขที่, จำนวนเงิน ฯลฯ) และ **รวมจำนวนเงินทั้งสิ้น** ด้านล่าง
3. คลิกปุ่ม **"พิมพ์สรุปรายงาน"** จะเปิดหน้าต่างพิมพ์ของเบราว์เซอร์
4. เลือกเครื่องพิมพ์หรือ "Save as PDF" ตามต้องการ แล้วกดพิมพ์/บันทึก

## ส่วนที่ 7: แก้ปัญหาเบื้องต้น

### ปัญหา: ดับเบิลคลิก run.bat แล้วหน้าต่างปิดทันที หรือขึ้นว่า "python ไม่รู้จัก"

- **สาเหตุ:** Python ยังไม่อยู่ใน PATH หรือยังไม่ได้ติดตั้ง
- **แก้:** ติดตั้ง Python ใหม่ และติ๊ก **"Add python.exe to PATH"** จากนั้นปิด Command Prompt เก่าทั้งหมดแล้วลองใหม่

### ปัญหา: เปิด http://localhost:5000 แล้วเข้าไม่ได้

- ตรวจสอบว่าหน้าต่างที่รัน **run.bat** หรือ **python backend\app.py** ยังเปิดอยู่และไม่มีข้อความ error สีแดง
- ลองพิมพ์ **http://127.0.0.1:5000** แทน

### ปัญหา: อัพโหลดรูปแล้วไม่เห็นข้อมูลดึงมาจากรูป

- ถ้าไม่ได้ติดตั้ง Tesseract หรือภาษาไทย จะดึงข้อความจากรูปไม่ได้ เป็นเรื่องปกติ ให้ไปที่ "รายการทั้งหมด" แล้วคลิก "แก้ไข" เพื่อกรอกข้อมูลเอง
- ถ้าติดตั้ง Tesseract แล้วยังดึงไม่ได้ รูปอาจเบลอหรือแสงไม่ดี ลองถ่ายใหม่หรือกรอกมือในฟอร์มแก้ไข

### ปัญหา: pip install -r requirements.txt แล้ว error

- ลองใช้: `python -m pip install -r requirements.txt`
- หรืออัปเกรด pip ก่อน: `python -m pip install --upgrade pip` แล้วค่อยรัน `pip install -r requirements.txt` อีกครั้ง

### ปัญหา: เปิดในมือถือไม่ได้ (เข้าระบบบนมือถือไม่ได้)

- ตรวจสอบว่า **มือถือและคอมอยู่ใน WiFi เดียวกัน** (ไม่ใช้ข้อมูลมือถือ)
- ตรวจสอบว่าได้ใส่ **IP ถูกต้อง** ของคอม (ดูจาก `ipconfig` บนคอม)
- บน Windows: ไปที่ **Settings → Network & Internet → Windows Firewall** ตรวจสอบว่าพอร์ต 5000 ไม่ถูกบล็อก หรือลองปิด firewall ชั่วคราวเพื่อทดสอบ
- บนคอม ต้อง**รันระบบอยู่** (run.bat หรือ python backend\app.py ยังเปิดอยู่)

### ปัญหา: ใช้ ngrok แล้วเจอหน้า "Visit Site" หรือ "You are about to visit..."

- ngrok ฟรีจะแสดงหน้าเตือนก่อนเข้า — ให้คลิกปุ่ม **"Visit Site"** หรือ **"Continue"** เพื่อเข้าใช้งานได้
